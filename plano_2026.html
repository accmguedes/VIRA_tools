Como usar
  • Clica nos títulos das frentes e blocos para abrir/fechar.
  • Tudo é editável: nome de frente, bloco, tema e tarefa. Clica no texto para editar, clica fora para guardar (ou tecla Enter).
  • Reordenar (arrastar): passa o rato sobre qualquer item — aparece "⋮⋮" à esquerda. Agarra aí e arrasta para a posição desejada. Funciona para tarefas, temas, blocos e frentes inteiras. Linhas podem mover-se entre temas, blocos e frentes diferentes.
  • Reordenar (setas): em alternativa ao arrasto, cada tarefa tem setas ▲ ▼ para mover uma posição de cada vez (útil em tablet/telemóvel).
  • Marca o checkbox para concluir; passa pelo botão de status para alternar entre Em curso / Bloqueado / Por fazer / Feito.
  • Clica em "+ trim" numa tarefa para a posicionar no Gantt — escolhe trimestre de início e duração.
  • Adicionar: botões "+ tarefa", "+ tema", "+ bloco / fase", "+ frente" para acrescentar.
  • Apagar: passa o rato sobre qualquer título — aparece um × à direita.
  • ↑ Publicar: envia o estado actual para o GitHub. A equipa vê a versão publicada (não a tua). Configura uma vez em ⚙.
  • Exportar/Importar guardam e carregam o estado num ficheiro JSON. Faz isto para backup ou para passar entre computadores.
  • Backup automático: sempre que importas um ficheiro, o estado actual é guardado num backup. Clica em "↶ Backup" para recuperar o último.
  • As alterações ficam guardadas neste browser. Se mudares de máquina, exporta antes.
